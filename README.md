My DX12 Game Engine
基于 DirectX12 API 的自研轻量级游戏引擎

## 项目介绍
这是一款基于 DirectX 12 图形 API 开发的轻量级游戏引擎，目前处于学习与迭代阶段，仅用于学习、研究与技术交流。
项目学习与参考自 The Cherno 的游戏引擎开发教程，在此向 The Cherno 致以诚挚的感谢。

A lightweight game engine based on the DirectX 12 API. Currently in the learning and development stage, for study and technical discussion only.
This project follows and references The Cherno's Game Engine Development series, with sincere thanks to The Cherno.

## 项目结构
- 引擎框架：Hazel
- 测试示例：Sandbox
- 图形 API：DirectX 12
- 开发状态：持续学习更新中

## 用途
仅供学习游戏引擎开发、图形学研究与技术交流使用。
For learning game engine development, graphics research, and technical discussion only.
